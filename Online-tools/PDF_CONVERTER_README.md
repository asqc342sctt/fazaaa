# 📄 أداة محول PDF - FazzaPlay

## نظرة عامة
أداة محول PDF محسنة ومتطورة تتيح للمستخدمين تحويل النصوص والملفات إلى تنسيق PDF بسهولة وسرعة. تم تطوير هذه الأداة لتعمل بشكل مثالي على جميع الأجهزة والهواتف المحمولة.

## ✨ المميزات الرئيسية

### 🔄 أنواع التحويل المدعومة
1. **نص إلى PDF** - تحويل النصوص العادية إلى PDF
2. **HTML إلى PDF** - تحويل كود HTML إلى PDF
3. **ملف إلى PDF** - تحويل الملفات النصية (TXT, HTML, CSS, JS) إلى PDF

### ⚙️ إعدادات PDF متقدمة
- **حجم الصفحة**: A4, Letter, Legal, A3
- **اتجاه الصفحة**: عمودي أو أفقي
- **حجم الخط**: 12, 14, 16, 18 بكسل
- **تخطيط تلقائي**: تقسيم النص تلقائياً على الصفحات

### 🎨 واجهة مستخدم محسنة
- تصميم متجاوب بالكامل
- دعم الوضع المظلم
- رسوم متحركة سلسة
- إشعارات تفاعلية
- أزرار سهلة الاستخدام

## 🚀 كيفية الاستخدام

### 1. اختيار نوع التحويل
- اختر "نص إلى PDF" لتحويل النصوص العادية
- اختر "HTML إلى PDF" لتحويل كود HTML
- اختر "ملف إلى PDF" لتحويل الملفات

### 2. إدخال المحتوى
- **للنص**: اكتب النص في مربع النص
- **للملف**: اختر ملف من جهازك
- **لـ HTML**: اكتب كود HTML في مربع النص

### 3. تخصيص الإعدادات
- اختر حجم الصفحة المناسب
- حدد اتجاه الصفحة
- اختر حجم الخط المفضل

### 4. التحويل والتحميل
- اضغط على "تحويل إلى PDF"
- انتظر حتى يكتمل التحويل
- اضغط على "تحميل PDF" لتنزيل الملف
