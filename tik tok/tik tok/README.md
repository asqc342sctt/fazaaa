# FazzaPlay - تنزيل فيديوهات تيك توك

واجهة عربية احترافية مع خادم Node/Express يقوم بجلب روابط التحميل من واجهة موثوقة عبر RapidAPI.

## المتطلبات
- Node.js 18+
- حساب على RapidAPI ومفتاح `RAPIDAPI_KEY`

## الإعداد السريع
1) انسخ الملف `.env.example` إلى `.env` وعدل القيم:

```
RAPIDAPI_KEY=ضع-مفتاح-RapidAPI-الخاصة-بك-هنا
PORT=3000
```

2) تثبيت الحزم:

```
npm install
```

3) التشغيل في وضع التطوير:

```
npm run dev
```

ثم افتح المتصفح على: `http://localhost:3000`

## ملاحظات
- إذا لم يتم وضع مفتاح `RAPIDAPI_KEY` سيعمل الموقع، لكن طلبات التحميل ستُرجع رسالة خطأ توضح نقص الإعداد.
- لا نقوم بتخزين الفيديوهات. الخدمة تُعيد الروابط مباشرة للمستخدم.
